ToolBox 是一个多功能平台，旨在托管丰富的插件生态系统，将强大的工具无缝集成到您的日常工作中。作为该生态系统的扩展，**文件管理插件**为您提供了一个功能强大的文件浏览器，支持系统文件浏览、常用路径收藏和拖拽文件复制。

## 什么是文件管理插件？

文件管理插件是 ToolBox 的一个文件浏览器工具，提供直观的三栏布局界面，左侧显示系统路径和常用收藏，中间展示当前目录文件列表，右侧管理目标路径。支持按类型筛选文件、搜索文件、拖拽复制文件等功能。

## 主要功能

- **三栏布局：** 左侧系统路径/常用路径、中间文件列表、右侧目标路径，清晰的文件管理体验。
- **路径导航：** 支持点击常用路径快速跳转，点击面包屑导航返回上级目录。
- **文件筛选：** 按类型筛选文件（图片、文档、表格、视频、文件夹），支持全选/单选。
- **文件搜索：** 实时搜索当前目录下的文件，支持模糊匹配。
- **常用路径：** 收藏常用文件夹，快速访问，支持添加和移除。
- **目标路径：** 管理目标路径列表，拖拽文件到目标路径卡片即可复制。
- **文件操作：**
  - 双击打开文件或文件夹
  - 右键菜单：打开、添加到常用、设为目标路径、删除
- **拖拽复制：** 拖拽文件到右侧目标路径卡片，快速完成文件复制。
- **面板宽度调整：** 拖动分隔线调整左右面板宽度，自动保存布局。
- **暗色模式：** 完整支持亮色/暗色主题切换。

## 配置与使用

1. **安装：** 从 ToolBox 插件商店安装 `文件管理` 插件。
2. **浏览文件：** 点击左侧系统路径或常用路径，中间区域显示文件列表。
3. **搜索文件：** 在顶部搜索框输入关键词，实时过滤文件列表。
4. **筛选类型：** 使用类型筛选器过滤特定类型的文件（图片、文档、表格、视频、文件夹）。
5. **添加常用路径：** 右键点击文件夹，选择"添加到常用"。
6. **设置目标路径：** 右键点击文件夹，选择"设为目标路径"，或点击右侧"添加路径"按钮手动添加。
7. **拖拽复制：** 从中间文件列表拖拽文件到右侧目标路径卡片，自动复制文件。
8. **删除文件：** 右键点击文件或文件夹，选择"删除"（需确认）。
